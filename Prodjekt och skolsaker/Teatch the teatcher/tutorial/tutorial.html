Hur man animerar i CSS

För att skapa animationer i CSS behöver du förstå tre grundläggande koncept:

  1. Keyframes: Används för att definiera de olika stegen i en animation.
  2. Animation-egenskapen: En kortform som kombinerar alla animationsegenskaper till en enda deklaration.
  3. Transition-egenskapen: Används för att skapa mjuka övergångar mellan olika stilar.

Genom att bemästra dessa tre koncept kan du skapa en mängd olika animationer, från enkla övertoningar och glidningar till komplexa sekvenser och effekter.

Här är en kort översikt över varje koncept:

Keyframes

Keyframes används för att definiera de olika stegen i en animation. Du kan specificera stiländringar vid olika punkter i animationssekvensen med hjälp av procent eller nyckelord som from och to.

Animation-egenskapen

Animation-egenskapen är en kortform som låter dig kombinera alla animationsegenskaper till en enda deklaration. Du kan specificera keyframe-animationens namn, varaktighet, timing-funktion, fördröjning, antal upprepningar, riktning, fyllningsläge och spelstatus i en rad.

Transition-egenskapen

Genom att kombinera dessa tre koncept kan du skapa dynamiska och engagerande animationer som förbättrar användarupplevelsen på din webbplats.

Hur man använder keyframes

Keyframes används för att definiera själva animationen. De specificerar stiländringar vid olika punkter i animationssekvensen.

För att skapa en keyframe-animation använder du regeln @keyframes. Inuti regeln @keyframes definierar du stilarna för olika punkter i animationen, vanligtvis med procent för att indikera tiden.

Till exempel:


                @keyframes example {
                    0% { background-color: red; }
                    50% { background-color: yellow; }
                    100% { background-color: green; }
                }

I detta exempel kommer bakgrundsfärgen att börja som röd, ändras till gul vid halvvägs och sluta som grön. Exemple finns till höger->

För att tillämpa animationen på ett element använder du egenskapen animation:


                .animated-element {
                    animation: exempel 4s infinite;
                    width: 300px;
                    max-height: 500px;
                }

Detta kommer att tillämpa animationen example på elementet med en varaktighet på 5 sekunder, och den kommer att upprepas oändligt. Den har även dimentionerna width: 300px och max-height: 500px.

Du kan också specificera ytterligare egenskaper för animationen, såsom timing-funktion, fördröjning och antal upprepningar. Till exempel:


                .animated-element {
                animation: example 5s ease-in-out 1s 3;
                }

I detta exempel kommer animationen att ha en ease-in-out timing-funktion, en fördröjning på 1 sekund innan den startar, och den kommer att upprepas 3 gånger. För att se exemple till höger "Refresha" sidan.


Hur man använder animation-egenskapen

Egenskapen animation är en kortform som kombinerar alla animationsegenskaper till en enda deklaration. Den låter dig specificera keyframe-animationens namn, varaktighet, timing-funktion, fördröjning, antal upprepningar, riktning, fyllningsläge och spelstatus i en rad.

Till exempel:


                .animated-element {
                    animation: newAnimation 6s linear 2s infinite alternate-reverse both running;
                }

Detta kommer att tillämpa animationen newAnimation på elementet med en varaktighet på 6 sekunder, en linjär timing-funktion, en fördröjning på 2 sekunder innan den startar, oändliga upprepningar, en alternerande omvänd riktning, både fyllningsläge och en spelstatus som körs.

Här är en uppdelning av egenskaperna som används i kortformen:

  • newAnimation: Namnet på keyframe-animationen.
  • 6s: Varaktigheten av animationen (6 sekunder).
  • linear: Timing-funktionen för animationen.
  • 2s: Fördröjningen innan animationen startar (2 sekunder).
  • infinite: Antalet gånger animationen kommer att upprepas (oändligt).
  • alternate-reverse: Riktningen för animationen (alternerar mellan framåt och bakåt, omvänt).
  • both: Fyllningsläget för animationen (tillämpas stilarna både i början och slutet av animationen).
  • running: Spelstatusen för animationen (startar i körande läge).

Du kan också använda de individuella animationsegenskaperna för att uppnå samma effekt. Till exempel:


            .animated-element {
                animation-name: newAnimation;
                animation-duration: 6s;
                animation-timing-function: linear;
                animation-delay: 2s;
                animation-iteration-count: infinite;
                animation-direction: alternate-reverse;
                animation-fill-mode: both;
                animation-play-state: running;
            }

Detta kommer att ge samma resultat som kortformen.

Du kan också kontrollera animationens spelstatus dynamiskt med JavaScript. Till exempel kan du starta, pausa eller starta om animationen baserat på användarinteraktioner:


                // Kod för att starta animationen för #animated-element när sidan laddas och när elementet klickas
                document.addEventListener('DOMContentLoaded', (event) => {
                    const animated-element = document.querySelector('#animated-element');
                
                    // Lägg till en klickhändelse för att starta animationen
                    animated-element.addEventListener('click', () => {
                        animated-element.style.animationPlayState = 'running';
                    });
                });

I detta exempel kan animationen från något med id="animated-element" kontrolleras med JavaScript för att starta, pausa eller starta om baserat på användarinteraktioner eller andra händelser.

Hur man använder transition-egenskapen

Egenskapen transition används för att skapa mjuka övergångar mellan olika stilar. Den låter dig specificera egenskapen du vill övergå, varaktigheten av övergången, timing-funktionen och fördröjningen innan övergången startar.

Till exempel:


        .transition-element {
            transition: background-color 1s ease-in-out 0.5s;
        }

Detta kommer att skapa en övergångseffekt på egenskapen background-color för elementet med en varaktighet på 1 sekund, en ease-in-out timing-funktion och en fördröjning på 0.5 sekunder innan den startar.

Du kan också specificera flera egenskaper att övergå, separerade med kommatecken. Till exempel:


        .transition-element {
            transition: background-color 1s ease-in-out 0.5s, color 1s ease-in-out 0.5s;
        }

Detta kommer att skapa en övergångseffekt på både egenskaperna background-color och color för elementet.

Som standard kommer övergången att ske när egenskapen ändras. Du kan också utlösa övergången med JavaScript genom att ändra egenskapsvärdet dynamiskt:


        // Kod för att starta #transitions-element när den klickas
        document.addEventListener('DOMContentLoaded', (event) => {
            const element = document.querySelector('#transition-element');
        
            element.addEventListener('click', () => {
                // Ändra bakgrundsfärgen till blå
                element.style.backgroundColor = 'blue';
        
                // Återställ bakgrundsfärgen till röd efter 5 sekunder
                setTimeout(() => {
                    element.style.backgroundColor = 'red';
                }, 5000);
            });
        });

I detta exempel kommer övergångseffekten att ske när man klickar på elementet och då blir elementet blå och gå tillbaka till röd efter 5 sekunder.

Övergångar är ett utmärkt sätt att lägga till subtila animationer på din webbplats utan behov av keyframes eller komplexa animationer. De kan användas för att skapa hovringseffekter, knappanimationer och mer.


Sammanfattning och Nästa Steg

Att animera element i CSS kan lägga till visuellt intresse och interaktivitet på din webbplats. Genom att använda keyframes, animation-egenskapen och transition-egenskapen kan du skapa en mängd olika animationer, från enkla övertoningar och glidningar till komplexa sekvenser och effekter.

Experimentera med olika egenskaper, värden och timing-funktioner för att skapa unika animationer som förbättrar användarupplevelsen och får din webbplats att sticka ut.

Kom ihåg att testa dina animationer på olika enheter och webbläsare för att säkerställa att de fungerar korrekt och ger en konsekvent upplevelse för alla användare.

Med övning och experimentering kan du bli skicklig på att animera element i CSS och skapa fantastiska animationer som fängslar din publik och ger liv åt din webbplats.

För att ta dina kunskaper till nästa nivå, överväg att utforska JavaScript-bibliotek som GSAP eller Anime.js för ännu mer avancerade animationer och interaktiva effekter.
